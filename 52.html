centos7 安装与配置 redis

2016/1/4 posted in  python

一、安装

命令yum install redis安装很方便

二、初始化常用配置

yum安装后默认配置文件在/etc/redis.conf

使用daemon方式运行程序，默认为非daemon方式运行
daemonize yes

为了速度我挂载了固态硬盘来为redis做持久化，所以要配置下持久化的路径
dbfilename dump.rdb #SNAPSHOTTING的文件名
dir /opt/data/redis/ #SNAPSHOTTING文件的路径

设置为警告日志记录，减少日质量
loglevel warning

注释掉本地IP绑定，开启外网访问
# bind 127.0.0.1

添加requirepass设置连接密码
requirepass ***（你的密码）

三、编写启动脚本

输入命令
vim /etc/init.d/redis
脚本代码：
```shell
#chkconfig: 2345 10 90
#description: Start and Stop redis
PATH=/usr/local/bin:/sbin:/usr/bin:/bin #根据你的本机情况设定

REDISPORT=6379
EXEC=/usr/local/bin/redis-server #根据你的本机情况设定
REDIS_CLI=/usr/local/bin/redis-cli #根据你的本机情况设定

PIDFILE=/var/run/redis.pid #根据你的本机情况设定（这里要和redis.conf设置一致）
CONF=/etc/redis.conf #根据你的本机情况设定

case $1 in
start)
if [ -f \(PIDFILE ]
then
echo "\)PIDFILE exists, process is already running or crashed
else
echoStarting Redis server...
\(EXEC \)CONF
fi
if [\(?"="0" ]
then
echo "Redis is running..."
fi
;;
stop)
if [ ! -f \)PIDFILE ]
then
echo $PIDFILE does not exist, process is not running
else
PID=$(cat \(PIDFILE)
echo "Stopping ..."
\$REDIS_CLI -p \)REDISPORT SHUTDOWN
while [ -x \({PIDFILE} ]
do
echo "Waiting for Redis to shutdown ..."
sleep 1
done
echo "Redis stopped"
fi
;;
restart|force-reload)
\){0} stop
${0} start
;;
*)
echo Usage: /etc/init.d/redis {start|stop|restart|force-reload} >&2
exit 1
esac
``
给启动脚本设定权限
chmod +x /etc/init.d/redis`

设置开机启动服务(centos 6.5)
sudo chkconfig redis on

设置开机启动服务(centos 7)
vim /etc/rc.d/rc.local

设置权限否则不会生效
chmod +x /etc/rc.d/rc.local

到这里就配置完成了

启动服务：
service redis start

停止服务：
service redis stop
